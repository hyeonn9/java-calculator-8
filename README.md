## 기능 요구 사항
입력한 문자열에서 구분자를 기준으로 숫자를 추출하여 더하는 계산기를 구현

- 기본 구분자: 쉼표(,)와 콜론(:)
    - `"" => 0`
    - `"1,2" => 3`
    - `"1,2:3" => 6`
- 커스텀 구분자: 문자열 앞부분에 "//"와 "\n" 사이에 위치하는 문자
    - `//;\n1;2;3 => 6`
- 사용자가 잘못된 값을 입력할 경우, `IllegalArgumentException`을 발생시킨 후 애플리케이션은 종료

## 입출력 요구 사항
### 입력
구분자와 양수로 된 문자열
```
덧셈할 문자열을 입력해주세요.
1,2:3    
```
### 출력
덧셈 결과
```
결과: 6
```
